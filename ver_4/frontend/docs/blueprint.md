# **App Name**: AuditX

## Core Features:

- Audio Playback: Play and control call audio with a waveform timeline.
- Transcript Viewer: Display synchronized multi-speaker transcripts with speaker labels and color-coding.
- Flagged Event Detection: Highlight AI-flagged compliance issues in the transcript and timeline.
- Evidence Explorer: Show the AI reasoning behind each flag with confidence scores, highlighted keywords, and rule matches. Uses tool reasoning to explain the relevance of supporting evidence.
- Session Export: Export session data including transcripts, flags, and annotations in various formats.
- Session Tagging: Tag sessions and flags with custom notes and status markers for review and feedback.
- Search and Filter: Enable text search and filtering of transcripts to quickly find specific keywords or flagged event types.

## Style Guidelines:

- Primary background: Light Gray (#F2F4F7) for the main workspace.
- Base Text/Titles: Charcoal Navy (#172B4D) for high contrast on a light background.
- Secondary Text/UI: Slate Gray (#6B7280) for secondary information.
- Primary Accent: Teal Blue (#205375) for a trustworthy tech feel.
- Secondary Accent: Muted Mint (#56B7C9) for buttons/links.
- Alert/Flag Color: Vibrant Orange (#E68A00) for flagged events.
- Body and headline font: 'Inter', a grotesque-style sans-serif for a modern, objective look.
- Use simple, clear icons from HeroIcons to represent various features and flags.
- Two-pane layout with audio and timeline on the left, transcript and details on the right.
- Subtle animations for loading states, flag highlights, and UI transitions.
